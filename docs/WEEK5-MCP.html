Week 5: MCP Integration

What is MCP?

Model Context Protocol is a standard for connecting AI assistants to external tools and data.

┌─────────────────┐ ┌─────────────────┐ │ Claude Desktop │ │ Cursor │ │ or other AI │ │ IDE │ └────────┬────────┘ └────────┬────────┘ │ │ │ MCP Protocol │ └───────────┬───────────────┘ │ ▼ ┌─────────────────────┐ │ MCP Server │ │ (your RAG system) │ │ │ │ Tools: │ │ - search_patients │ │ - query_notes │ │ - get_patient │ └─────────────────────┘
Why MCP? Instead of building a custom UI, you can expose your RAG system as tools that any MCP-compatible AI can use. Claude, Cursor, and other tools can then query your medical records naturally.

MCP Concepts

Concept Description Example
Server Your application that provides tools Medical RAG server
Client AI assistant that calls your tools Claude Desktop, Cursor
Tools Functions the AI can call search_patients, query_notes
Resources Data the AI can read Patient records, documents

Setting Up an MCP Server

1. Install the SDK

npm install @modelcontextprotocol/sdk

2. Create the server

// mcp-server/index.ts
import { McpServer } from '@modelcontextprotocol/sdk/server/mcp.js';
import { StdioServerTransport } from '@modelcontextprotocol/sdk/server/stdio.js';

const server = new McpServer({
  name: 'medical-rag',
  version: '1.0.0',
});

// Register tools (we'll add these next)

// Start the server
const transport = new StdioServerTransport();
await server.connect(transport);

Defining Tools

// Tool: Search for patients
server.tool(
  'search_patients',
  'Search for patients by name or condition',
  {
    query: {
      type: 'string',
      description: 'Search query (name or condition)',
    },
    limit: {
      type: 'number',
      description: 'Max results to return',
      default: 10,
    },
  },
  async ({ query, limit }) => {
    // Call your existing RAG system
    const results = await executeQuery(query, { limit });

    return {
      content: [
        {
          type: 'text',
          text: formatResultsForLLM(results),
        },
      ],
    };
  }
);

// Tool: Query clinical notes
server.tool(
  'query_notes',
  'Search clinical notes semantically',
  {
    query: {
      type: 'string',
      description: 'Semantic search query',
    },
    patientId: {
      type: 'string',
      description: 'Optional: limit to specific patient',
      optional: true,
    },
  },
  async ({ query, patientId }) => {
    const results = await searchClinicalNotes(query, {
      patientIds: patientId ? [patientId] : undefined,
    });

    return {
      content: [
        {
          type: 'text',
          text: formatVectorResults(results),
        },
      ],
    };
  }
);

// Tool: Get patient details
server.tool(
  'get_patient',
  'Get full details for a specific patient',
  {
    patientId: {
      type: 'string',
      description: 'The patient ID',
    },
  },
  async ({ patientId }) => {
    const patient = await getPatientById(patientId);

    return {
      content: [
        {
          type: 'text',
          text: formatPatientDetails(patient),
        },
      ],
    };
  }
);

Configuring Claude Desktop

Add to ~/Library/Application Support/Claude/claude_desktop_config.json:

{
  "mcpServers": {
    "medical-rag": {
      "command": "npx",
      "args": ["ts-node", "/path/to/medical-rag/mcp-server/index.ts"],
      "env": {
        "DATABASE_URL": "your-neon-url",
        "PINECONE_API_KEY": "your-pinecone-key",
        "OPENAI_API_KEY": "your-openai-key"
      }
    }
  }
}

Restart Claude Desktop

After saving the config, restart Claude Desktop. You should see your tools available.

Configuring Cursor

Add to .cursor/mcp.json in your project:

{
  "mcpServers": {
    "medical-rag": {
      "command": "npx",
      "args": ["ts-node", "mcp-server/index.ts"]
    }
  }
}

How It Works

Call query_notes(query="foot pain", patientId=...) │ └─────────────────────────────────────────────────────────────┘ │ ▼ ┌─────────────────────────────────────────────────────────────┐ │ MCP Server │ │ │ │ - Receives tool calls │ │ - Executes RAG queries │ │ - Returns formatted results │ └─────────────────────────────────────────────────────────────┘ │ ▼ ┌─────────────────────────────────────────────────────────────┐ │ Claude Desktop │ │ │ │ "I found 3 diabetic patients with notes about foot pain: │ │ - John Smith: noted tingling in feet on 2024-01-15 │ │ - Jane Doe: foot ulcer assessment on 2024-02-20 │ │ ..." │ └─────────────────────────────────────────────────────────────┘
