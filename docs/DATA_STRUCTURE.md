# Synthea Coherent Dataset Structure

## Overview

The Coherent dataset is a synthetic FHIR R4 dataset generated by [Synthea](https://github.com/synthetichealth/synthea), designed to provide realistic medical records with clinical notes (SOAP-style documentation).

**Source**: `s3://synthea-open-data/coherent/unzipped/fhir/`
**Size**: ~4.4GB, 1,278 patient bundles
**Format**: FHIR R4 JSON bundles

## File Structure

```
data/coherent/fhir/
├── Abraham100_Stracke611_abc123.json    # Each file = one patient's complete history
├── Abigail123_Johnson456_def789.json
└── ... (1,278 patient files)
```

Each JSON file is a **FHIR Bundle** containing all resources for a single patient.

## Bundle Structure

```json
{
  "resourceType": "Bundle",
  "type": "collection",
  "entry": [
    { "resource": { "resourceType": "Patient", ... } },
    { "resource": { "resourceType": "Condition", ... } },
    { "resource": { "resourceType": "Observation", ... } },
    { "resource": { "resourceType": "MedicationRequest", ... } },
    { "resource": { "resourceType": "Procedure", ... } },
    { "resource": { "resourceType": "Encounter", ... } },
    { "resource": { "resourceType": "Immunization", ... } },
    { "resource": { "resourceType": "AllergyIntolerance", ... } },
    { "resource": { "resourceType": "DocumentReference", ... } }
  ]
}
```

## Resource Types

### 1. Patient (1 per file)

Demographics and identification.

```json
{
  "resourceType": "Patient",
  "id": "abc-123-def",
  "name": [{ "given": ["John"], "family": "Smith" }],
  "gender": "male",
  "birthDate": "1945-03-15",
  "address": [{
    "city": "Boston",
    "state": "Massachusetts",
    "postalCode": "02101"
  }]
}
```

**Typical count**: 1 per patient

### 2. Condition (Diagnoses)

Medical conditions and diagnoses using SNOMED-CT codes.

```json
{
  "resourceType": "Condition",
  "id": "cond-123",
  "code": {
    "coding": [{
      "system": "http://snomed.info/sct",
      "code": "44054006",
      "display": "Diabetes mellitus type 2"
    }]
  },
  "clinicalStatus": { "coding": [{ "code": "active" }] },
  "onsetDateTime": "2010-05-20"
}
```

**Typical count**: 5-30 per patient
**Common conditions**: Hypertension, Diabetes, Obesity, COPD, Depression, Heart Failure

### 3. Observation (Labs & Vitals)

Lab results and vital signs using LOINC codes.

```json
{
  "resourceType": "Observation",
  "id": "obs-456",
  "code": {
    "coding": [{
      "system": "http://loinc.org",
      "code": "4548-4",
      "display": "Hemoglobin A1c/Hemoglobin.total in Blood"
    }]
  },
  "valueQuantity": {
    "value": 7.2,
    "unit": "%"
  },
  "effectiveDateTime": "2023-01-15",
  "referenceRange": [{
    "low": { "value": 4.0 },
    "high": { "value": 5.6 }
  }]
}
```

**Typical count**: 100-500+ per patient
**Common observations**: A1C, Glucose, Cholesterol, BMI, Blood Pressure, Heart Rate

### 4. MedicationRequest

Prescribed medications using RxNorm codes.

```json
{
  "resourceType": "MedicationRequest",
  "id": "med-789",
  "medicationCodeableConcept": {
    "coding": [{
      "system": "http://www.nlm.nih.gov/research/umls/rxnorm",
      "code": "860975",
      "display": "Metformin hydrochloride 500 MG Oral Tablet"
    }]
  },
  "status": "active",
  "intent": "order",
  "dosageInstruction": [{ "text": "Take 1 tablet twice daily" }],
  "authoredOn": "2023-01-15"
}
```

**Typical count**: 20-100 per patient

### 5. Procedure

Medical procedures performed using SNOMED-CT codes.

```json
{
  "resourceType": "Procedure",
  "id": "proc-abc",
  "code": {
    "coding": [{
      "system": "http://snomed.info/sct",
      "code": "430193006",
      "display": "Medication reconciliation"
    }]
  },
  "status": "completed",
  "performedDateTime": "2023-06-01"
}
```

**Typical count**: 10-50 per patient

### 6. Encounter (Visits)

Healthcare encounters and visits.

```json
{
  "resourceType": "Encounter",
  "id": "enc-xyz",
  "class": { "code": "ambulatory" },
  "type": [{ "text": "Outpatient visit" }],
  "status": "finished",
  "period": {
    "start": "2023-06-01T09:00:00Z",
    "end": "2023-06-01T09:30:00Z"
  }
}
```

**Typical count**: 50-200 per patient

### 7. Immunization

Vaccination records using CVX codes.

```json
{
  "resourceType": "Immunization",
  "id": "imm-123",
  "vaccineCode": {
    "coding": [{
      "system": "http://hl7.org/fhir/sid/cvx",
      "code": "140",
      "display": "Influenza, seasonal"
    }]
  },
  "status": "completed",
  "occurrenceDateTime": "2023-10-01"
}
```

**Typical count**: 10-40 per patient

### 8. AllergyIntolerance

Documented allergies and intolerances.

```json
{
  "resourceType": "AllergyIntolerance",
  "id": "allergy-456",
  "code": {
    "coding": [{
      "system": "http://snomed.info/sct",
      "code": "419511003",
      "display": "Propensity to adverse reactions to drug"
    }]
  },
  "clinicalStatus": { "coding": [{ "code": "active" }] },
  "criticality": "high"
}
```

**Typical count**: 0-5 per patient

### 9. DocumentReference (Clinical Notes) ⭐

**The key resource for RAG** - contains SOAP-style clinical notes.

```json
{
  "resourceType": "DocumentReference",
  "id": "doc-789",
  "type": {
    "coding": [{
      "system": "http://loinc.org",
      "code": "34117-2",
      "display": "History and physical note"
    }]
  },
  "date": "2023-06-01",
  "content": [{
    "attachment": {
      "contentType": "text/plain",
      "data": "IyBDaGllZiBDb21wbGFpbnQuLi4="  // Base64 encoded
    }
  }]
}
```

**Typical count**: 50-150 per patient

## How We Store This Data

### Neon PostgreSQL (Structured Data)
- **patients** - Demographics
- **conditions** - Diagnoses with SNOMED codes
- **observations** - Labs/vitals with values and units
- **medications** - Prescriptions with dosage
- **procedures** - Medical procedures
- **encounters** - Visit records
- **immunizations** - Vaccine records
- **allergies** - Allergy records
- **documents** - Metadata only (pinecone_id reference)

### Pinecone (Semantic Search)
- Clinical note full text (decoded from base64)
- Embedded using text-embedding-3-small (1536 dimensions)
- Metadata: patient_id, patient_name, date, document_type

## Code Systems

| System | URL | Used For |
|--------|-----|----------|
| SNOMED-CT | http://snomed.info/sct | Conditions, Procedures |
| LOINC | http://loinc.org | Observations, Document types |
| RxNorm | http://www.nlm.nih.gov/research/umls/rxnorm | Medications |
| CVX | http://hl7.org/fhir/sid/cvx | Immunizations |
| ICD-10 | http://hl7.org/fhir/sid/icd-10 | Some conditions |
